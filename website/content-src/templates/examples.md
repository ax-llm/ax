# Examples

Examples are grouped by what they prove and by whether they run against a live provider. Start with the curated paths for the production-shaped examples, then use the archive when you need low-level protocol or package coverage files.

## How To Read These

- `Provider API` examples make real HTTP calls and list the environment variables they normally need.
- `No key` entries in the archive are lower-level local checks, protocol adapters, or package coverage files.
- Generated language examples live under each package's `examples` directory.
- TypeScript examples live under `src/examples`.
